# UI font

`NotoSansSC-Regular-ScaleSword.ttf` and
`NotoSansSC-Bold-ScaleSword.ttf` are static weight subsets of the official
Noto Sans SC variable font created for the ScaleSword UI. Regular (400) is
used for body copy and Bold (700) is used for titles, buttons, and primary
HUD information. Both subsets contain the glyphs used by
`localization/translation.csv`, plus the Latin characters, digits and
punctuation required by the game UI.

Source: [Noto CJK](https://github.com/notofonts/noto-cjk)
